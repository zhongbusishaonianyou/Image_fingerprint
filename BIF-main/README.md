### usage

#### 1. Requirements

1. cmake
2. PCL
3. OpenCV
4. python
5. Eigen3

#### 2. Run
cd ~/catkin_ws/src
git clone https://github.com/zhongbusishaonianyou/Image_fingerprint.git
cd ..
catkin_make 
source devel/setup.bash
rosrun fingerprint binary_fingerprint
